# Medical & Care Products Catalog MVP

## Description
This is a simple web application to list medical and care products for home use, consumables, and elderly care items.  
The catalog will allow an admin to manage products and categories.

## Target Users
- **Admin:** Manage catalog items
- **Visitor:** Browse and view product information

## Main Features
1. Categorized product catalog (equipment, consumables, home & elderly care)
2. Admin login and basic management
3. Product listing with basic details
